# AutoStream AI Agent

## Overview
This project is a simple AI agent that interacts with users, answers pricing queries, and converts interested users into leads.

## Features
- Detects user intent (greeting, pricing, high intent)
- Retrieves plan details from a JSON file (RAG)
- Identifies interested users
- Collects user details step-by-step
- Captures leads using a mock function

## How It Works
1. User enters a message  
2. Agent detects intent  
3. Shows pricing if asked  
4. Starts lead collection if user is interested  
5. Saves user details after completion  

## Run the Project
python main.py

## Note
This project simulates a real-world AI sales agent workflow.
